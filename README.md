# Rialo Mission Simulator 🚀

- 🪙 Real WalletConnect integration
- 🕹️ Mini missions (click, trivia, drag & drop)
- 📜 Mission logs + 🏆 Leaderboard
- 💻 Built with HTML, CSS, and JavaScript

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- WalletConnect v2
- GitHub Pages (hosting)
